## Overview

An application where users can store recipes and track what ingredients they have. Users can also select a recipe and get a shopping list generated based on what they do not already have.
Future features will include the option to ask for a selection of recipes to be chosen for you, for as many days as you need and a shopping list will be generated.
Potentially the option to see recipes added by other users

## Tech Stack

- **Frontend:** React, Next.js, TailwindCSS
- **Backend:** Node.js, PostgreSQL
- **Testing:** Vitest
- **Other tools:** dotenv, cors, lucide-react

## Setup

1. Clone repository:

```
git clone <git@github.com:pepnochizzy/Pantry_tracker.git>
```

2. Install dependencies:

```
npm install
```

3. Create new .env file and configure database credentials
